# Landing Page Portfolio

A collection of professional landing page designs showcasing modern web development skills.

## Projects

### 1. CleanPro Landing Page
A corporate cleaning service landing page with professional design, featuring:
- Modern gradient hero section
- Service showcase with pricing
- Testimonials and trust indicators
- Responsive design with Tailwind CSS

**[Live Demo](projects/cleanpro-landing/index.html)**

### 2. CleanLife Tokyo
A clean, minimal landing page for a Tokyo-based cleaning service, featuring:
- Smooth scroll animations
- Interactive accordion FAQ section
- Mobile-first responsive design
- Modern Japanese typography

**[Live Demo](projects/cleanlife-tokyo/index.html)**

## Tech Stack

- HTML5
- CSS3
- Tailwind CSS
- Vanilla JavaScript
- Responsive Design

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/tthuan/demo.git
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   npx serve .
   ```
